Codex Usage Tray Windows package

Install:
Double-click Install-CodexUsageTray.cmd.

Alternative:
Right-click Install-CodexUsageTray.ps1 and select "Run with PowerShell".

The installer copies codex-usage-tray.exe to:
%LOCALAPPDATA%\Programs\CodexUsageTray

It also creates a Start Menu shortcut named "Codex Usage Tray" and launches the app.

Portable run:
Double-click codex-usage-tray.exe.
Keep codex-usage-tray.exe.manifest next to the exe.
